## Deploy to GitHub Pages (2 Steps)

### Step 1: Upload to GitHub
1. Go to [github.com](https://github.com) → Click **"New"**
2. Name: `xiyu-studio` → Make **Public** → **Create repository**
3. Click **"Add file"** → **"Upload files"**
4. Drag & drop **all files** from this folder
5. Click **"Commit changes"**

### Step 2: Enable GitHub Pages
1. Go to **Settings** (top right)
2. Click **"Pages"** (left sidebar)
3. Source: **Deploy from a branch**
4. Branch: **main** → Folder: **/ (root)**
5. Click **Save**

✅ **Wait 1-2 minutes** → Your site will be live at: `https://yourusername.github.io/xiyu-studio`

## If GitHub Pages Fails

Try **Netlify** (usually faster):
1. Go to [netlify.com](https://netlify.com)
2. Click **"New site from Git"**
3. Select your GitHub repo → **Deploy**
4. Done in 30 seconds
