================================================================================
DEPLOYMENT INSTRUCTIONS
================================================================================

Local Testing:
  1. Create a folder: xiyu-studio/
  2. Place all files in the same directory
  3. Open index.html in a web browser
  4. Test all interactive features

Web Server Deployment:
  1. Upload all files to your server in the same folder
  2. Ensure index.html is set as default document
  3. Verify .css and .js files are served with correct MIME types
  4. Set cache headers (e.g., 30 days for images, 1 week for HTML/CSS/JS)
  5. Enable gzip compression on server
  6. Test responsive design on mobile devices
  7. Verify all external links work

CDN / Static Hosting (Netlify, Vercel, etc.):
  1. Connect repository or upload files directly
  2. Set root as "public" or deploy folder
  3. Ensure build command is empty (no build needed)
  4. Deploy and test
